# Password-strength-checker-

A simple program that checks how strong a password is, based on length and character variety.

It checks for:
- Minimum length (8 characters)
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

Based on how many of these are present, it rates the password as Weak, Medium, or Strong.

## Why I made this

Most passwords get compromised because of simple, predictable choices, not advanced attacks. I wanted to understand what actually makes a password weak or strong, so I built this to check it based on real criteria.

## Files
- password_checker.py
- PasswordChecker.java

## How to run

Python: python password_checker.py

Java: javac PasswordChecker.java then java PasswordChecker
